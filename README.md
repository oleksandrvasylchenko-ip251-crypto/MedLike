## ✏️ РЕДАГУВАННЯ

### Змінити текст
Відкрийте `index.html` у текстовому редакторі (VS Code, Sublime, etc.) і відредагуйте текст безпосередньо.

### Змінити кольори
Відкрийте `styles/styles.css` і змініть значення у :root:
```css
:root {
    --primary: #09B9AA;      /* Змініть тут */
    --dark: #081726;         /* Змініть тут */
    --light-bg: #f9fafb;
}
```

### Додати нові фото
1. Поміщайте фото в папку `images/`
2. Додайте в HTML:
```html
<img src="images/your-photo.webp" alt="Опис">
```

### Змінити логотип
Замініть содержимое SVG у `<div class="logo">` або додайте звичайне зображення:
```html
<img src="images/logo.png" alt="МЕДЛАЙК" class="logo">
```

## 🐛 УСУНЕННЯ НЕПОЛАДОК

**Проблема:** Фото не завантажуються
- **Рішення:** Перевірте, що папка `images/` знаходиться в одній директорії з `index.html`

**Проблема:** Слайдер не працює
- **Рішення:** Переконайтеся, що файл `script/script.js` завантажений (див. консоль браузера)

**Проблема:** Стилі не застосовуються
- **Рішення:** Перевірте, що файли `styles/styles.css` та `styles/mobile.css` завантажені

**Проблема:** Форма не валідує
- **Рішення:** Використовуйте сучасний браузер (Chrome, Firefox, Safari, Edge)
